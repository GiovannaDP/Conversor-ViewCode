# Conversor-ViewCode
App utilizado para converter algumas medidas, porém feito em ViewCode.

As unidades utilizadas são:

  	•	Celsius - Fahrenheit
    •	Kg - Libra
    •	Km - Metro
    •	Real - Dólar
